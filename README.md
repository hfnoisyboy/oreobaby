### The Hard Method
```js
GET QR
$ apt update
$ apt install nodejs --fix-missing
$ pkg install git
$ git clone https://github.com/hfnoisyboy/oreobaby
$ cd oreobaby
$ chmod +x *
$ npm install @adiwajshing/baileys
$ npm install chalk
$ node niya.js
```
